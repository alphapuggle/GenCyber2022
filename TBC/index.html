TBC is a useful application for having one bookmark for all of your Google Classrooms.
To setup, please copy the class code for each classroom, https://classroom.google.com/u/0/c/*************** (the star portion) into the correct period area. If you don't have a classroom for a certain period, leave the text area blank or enter a custom URL. The Math and Science radio buttons will open Pearson or My HRW. Other classes that use these textbooks can be then be selected as that class and be used like that.

Warning: Periods already set will not be visible.
